## Run Locally

Clone the project

```bash
  git clone https://github.com/akenolol/verbose-bot
```

Go to the project directory

```bash
  cd verbose-bot
```

Install dependencies

```bash
  npm install
```

Start the Project

```bash
  npm run start
```
